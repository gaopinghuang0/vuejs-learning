Vuejs Directives

Directives help to handle data-binding, event-listening, conditional rendering, and so on. Therefore, in Javascript, there is no need to interact with DOM elements, which is nicely decoupled.

All directives of Vuejs start with "v-", such as v-if, v-for, v-model, v-on, v-bind, and so on.

It's very like AngularJS in which directives start with "ng-".

1. Data binding from js, without directives

{{ message }} 
    
      <div id="app">
        {{ message }}
      </div>
    
    
    
      var app = new Vue({
        el: '#app',
        data: {
          message: 'Hello Vue!',
        }
      })

2. Data binding through "v-model"

You entered: {{ message }}

    
      <div id="app-2">
        <p>You entered: {{ message }}</p>
        <input v-model="message">
      </div>
    
    
    
      var app2 = new Vue({
        el: '#app-2',
        data: {
          message: 'Hello Vue!',
        }
      })

3. Data binding plus computed property

You entered: {{ message }}
Computed reverse: {{ reversedMessage }}

    
      <div id="app-3">
        <p>You entered: {{ message }} <br> 
          Computed reverse: {{ reversedMessage }}</p>
        <input v-model="message">
      </div>
    
    
    
      var app3 = new Vue({
        el: '#app-3',
        data: {
          message: 'Hello Vue!',
        },
        computed: {
          // a computed getter
          reversedMessage: function () {
            // `this` points to the app3 instance
            return this.message.split('').reverse().join('')
          }
        }
      })

4. Data binding with methods

You entered: {{ message }}
Reverse by method: {{ reverseMessage() }}

To know the difference between computed property and methods, see this comparison. Moreover, computed property has extra benefits when it involves more than one variable. Once any one variable changes, the computed property will also update accordingly. This saves the effort to watch variables manually (see this example if you are interested.)

    
      <div id="app-4">
        <p>You entered: {{ message }}  <br> 
          Reverse by method: {{ reverseMessage() }}</p>
        <input v-model="message">
      </div>
    
    
    
      var app4 = new Vue({
        el: '#app-4',
        data: {
          message: 'Hello Vue!',
        },
        methods: {
          reverseMessage: function () {
            return this.message.split('').reverse().join('')
          }
        }
      })

5. Conditional rendering via "v-if" and "v-else"

Picked: {{picked}}
Content of A
Content of B
    
      <div id="app-5">
        <div>
          <label>
            <input type="radio" value="A" v-model="picked"> Show A
          </label>
          <label>
            <input type="radio" value="B" v-model="picked"> Show B
          </label>
        </div>
        <div>
          Picked: {{picked}}
          <div v-if="picked === 'A'">
            <div>Content of A</div>
          </div>
          <div v-else>
            <div>Content of B</div>
          </div> 
        </div>
      </div>
    
    
    
      var app5 = new Vue({
        el: '#app-5',
        data: {
          picked: '',
        }
      })

6. Render a list of data via "v-for"

This is very helpful to generate a list of items that have the same format but different content.

  • {{ item.message }} 
    
      <div id="app-6">
        <ul id="example-1">
          <li v-for="item in items">
            {{ item.message }}
          </li>
        </ul>
      </div>
    
    
    
      var app6 = new Vue({
        el: '#app-6',
        data: {
          items: [
            {message: 'Foo'},
            {message: 'Bar'}
          ],
        }
      })

7. Event handling via "v-on"

Click to update counter: 
    
      <div id="app-7">
        Click to update counter:
        <div class="counter-group">
          <button v-on:click="decrease">-</button>
          <input type="text" v-model="counter" readonly="true">
          <button v-on:click="increase">+</button>
        </div>
      </div>
    
    
    
      var app7 = new Vue({
        el: '#app-7',
        data: {
          counter: 0,
        },
        methods: {
          decrease: function() {
            if (this.counter > 0) {
              this.counter -= 1
            }
          },
          increase: function() {
            this.counter += 1
          }
        }
      })
